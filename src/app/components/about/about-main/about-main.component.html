Sobre la Aplicación

Presentamos Aurum RH un registro de solicitantes y entrevistas técnicas, que se realizan durante el proceso de selección de un candidato. Los aspirantes al puesto y los usuarios de recursos humanos tendrán un registro de las ofertas de trabajo, de las entrevistas y de las pruebas técnicas. También se podrá mostrar el resultado, y la fecha de realización, de las habilidades que se han evaluado en las entrevistas técnicas. La aplicación también constará de una barra de progreso que muestre el punto del proceso donde se encuentra el candidato y un seguimiento del proceso del candidato.
